OVI Diseño sitios Web

OVI Diseño de sitios web

En la siguiente OVI podrá encontrar una introducción al diseño de sitios web, donde
se enfatiza en la importancia del lenguaje HTML, como funciona, la forma correcta de
utilizarlo y practicas de como generar sitios web de manera práctica e intuitiva con
el usuario final.



Objetivo general

Generar un apoyo en cuanto al diseño web donde se comprenda
como utilizar correctamente el lenguaje HTML para el desarrollo
de páginas web.



Objetivo específico 1

Aprender sobre el leguaje HTML, su correcto uso y sus ventajas.



Objetivo específico 2

Identificar la estructura correcta de uso para proyectar
en nuestro sitio web la idea deseada.



Objetivo específico 3

identificar los programas que se pueden utilizar para
escribir en lenguaje HTML.


Introducción a HTML

En su corazón, HTML es un lenguaje muy sencillo compuesto de elementos,
que pueden ser aplicados a piezas de texto para darles un significado
diferente en un documento (¿Esto es un párrafo? ¿Esto es una lista con
viñetas? ¿Esto es parte de una tabla?), estructura un documento en secciones
lógicas (¿Tiene una cabecera? ¿Tres columnas de contenido? ¿Un menú de
navegación?) e incrusta contenido como imágenes y vídeos en una página.
Este módulo introducirá los dos primeros de estos, e introduce conceptos
fundamentales y la sintaxis que necesitas para entender HTML.
